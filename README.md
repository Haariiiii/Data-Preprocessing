# Employee Data Preprocessing using Python

## Overview

This project demonstrates the complete data preprocessing workflow on an employee dataset using Python. The objective was to clean, transform, and prepare the dataset for machine learning and data analysis by handling missing values, encoding categorical variables, and scaling numerical features.

## Features

* Handled missing values using appropriate imputation techniques.
* Performed statistical analysis on numerical and categorical data.
* Renamed columns for better readability.
* Checked and handled duplicate records.
* Applied Label Encoding and One-Hot Encoding to categorical features.
* Performed feature scaling using **StandardScaler** and **MinMaxScaler**.
* Prepared the dataset for machine learning applications.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

## Preprocessing Steps

1. Loaded and explored the dataset.
2. Identified and handled missing values.
3. Performed descriptive statistical analysis.
4. Renamed columns where necessary.
5. Checked for duplicate records.
6. Encoded categorical variables using Label Encoding and One-Hot Encoding.
7. Scaled numerical features using StandardScaler and MinMaxScaler.
8. Generated the final cleaned dataset ready for analysis and machine learning.
